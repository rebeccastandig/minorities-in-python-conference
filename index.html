Welcome to the site for the Minorities in Python Conference!

About the Minorities in Python Conference

The Minorities in Python Conference is a new conference by and for anyone who identifies as a minority in the Python community. All are welcome! You don't need to be an expert programmer (or even know how to code yet!) in order to participate.

If you don't identify as a minority in the Python community, you are still welcome to volunteer or help in whatever way you desire.

This conference arose out of a need to have an inclusive, safe space to discuss minority-specific issues, ideas, and resources, as well as to bring together minorities in the Python community for networking, learning, sharing, and friendship. To that end, we ask that you adhere to our Code of Conduct (coming soon).

The first day of the Minorities in Python Conference will focus on teaching beginner and new programmers the basics of Python. Days 2 and 3 will consist of scheduled talks, lightning talks, and sprints.

How Can You Help?

We're in need of a venue, funds, childcare providers, volunteers...pretty much everything you need to put on a conference. If you're interested in helping to organize the Minorities in Python Conference, donate funds or services, or any other form of assistance, please email or tweet at the organizers.
